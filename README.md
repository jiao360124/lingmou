## 🔧 快速开始

### 环境要求

- **操作系统**: Windows 10/11, macOS 10.14+, Linux
- **内存**: 最低 2GB RAM, 推荐 4GB+
- **磁盘空间**: 最低 500MB, 推荐 1GB+

### 安装

```bash
# 克隆仓库
git clone https://github.com/jiao360124/AE8F88.git
cd AE8F88

# 配置环境
cp .env.example .env
cp .ports.env.example .ports.env

# 运行初始化
.\scripts\init.ps1
```

### 健康检查

```powershell
# 检查系统状态
.\scripts\simple-health-check.ps1

# 运行集成测试
.\scripts\integration-test.ps1 -Detailed

# 运行性能测试
.\scripts\performance-benchmark.ps1 -Detailed
```

## ⚙️ 配置

### 环境变量

所有配置通过环境变量管理：

```bash
# 基本配置
WORKSPACE_DIR=C:\Users\Administrator\.openclaw\workspace
TIMEZONE=Asia/Shanghai

# Gateway配置
GATEWAY_URL=http://localhost:8080
GATEWAY_TOKEN=your_token

# Git配置
GITHUB_REPO=https://github.com/jiao360124/AE8F88.git
GITHUB_TOKEN=your_token

# 日志配置
LOG_LEVEL=INFO
LOG_ROTATION_SIZE=10MB

# 备份配置
BACKUP_SCHEDULE=0 2 * * *
BACKUP_RETENTION_DAYS=7
```

### 端口配置

**文件**: `.ports.env`

```bash
GATEWAY_PORT=8080
API_PORT=8081
BACKUP_PORT=8082
```
