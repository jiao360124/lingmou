## 🚀 新增功能

### 1. AI/LLM工具包 (ai-toolkit)
整合了三个AI相关工具：
- **Prompt-Engineering** - 提示工程和质量检查
- **Moltbook社区集成** - AI Agent社区平台
- **RAG知识库** - 检索增强生成系统

### 2. 搜索工具包 (search-toolkit)
整合了两个搜索工具：
- **Exa Web Search** - AI驱动网络搜索
- **DeepWiki** - GitHub仓库文档查询

### 3. 开发工具包 (dev-toolkit)
整合了三个开发工具：
- **API Development** - REST/GraphQL API开发
- **Database Manager** - 多数据库管理
- **SQL Toolkit** - 关系型数据库操作

### 4. 核心架构优化
- **监控模块分组** - monitoring/目录
- **策略引擎分组** - strategy/目录
- **统一导出** - core/index.js

## 📖 使用指南

### 使用AI/LLM工具包
```bash
# 使用提示工程工具
pe templates --use code.function
pe quality --check "your prompt"
pe optimize --prompt "your prompt"

# 使用Moltbook
node ./skills/ai-toolkit/moltbook/scripts/api-client.ps1 -Action register -Name "灵眸"

# 使用RAG知识库
rag search "API调用"
rag add docs/api-guide.md --category documentation
```

### 使用搜索工具包
```bash
# Exa网络搜索
mcporter call 'exa.web_search_exa(query: "latest AI news 2026", numResults: 5)'

# Exa代码搜索
mcporter call 'exa.get_code_context_exa(query: "React hooks examples", tokensNum: 3000)'

# DeepWiki查询
node ./scripts/deepwiki.js ask cognitionlabs/devin "How do I use MCP?"
```

### 使用开发工具包
```bash
# API开发
curl -s https://api.example.com/users | jq .

# 数据库管理
"Connect to PostgreSQL database"
"Run query: SELECT * FROM users LIMIT 10"

# SQLite查询
sqlite3 mydb.sqlite "SELECT * FROM users WHERE created_at > '2026-01-01' LIMIT 10;"
```

### 使用核心模块
```javascript
// 加载核心模块
const core = require('./core');

// 初始化监控
core.monitoring.init({
  performance: { checkInterval: 60000 },
  memory: { warningThreshold: 0.75 },
  api: { checkInterval: 60000 }
});

// 使用策略引擎
const strategy = core.strategy.getEngine('enhanced');
const result = strategy.execute(task);

// 使用统一导出
const { monitoring, strategy, controlTower } = core;
```
