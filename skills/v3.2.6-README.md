## 🚀 快速开始

### 基本使用

```javascript
// 导入核心模块
const core = require('./core');

// 初始化
core.init({
  performance: { checkInterval: 60000 },
  memory: { checkInterval: 60000 },
  api: { trackInterval: 60000 }
});

// 使用监控
core.performanceMonitor.getCurrentMemoryUsage();

// 使用策略引擎
const strategy = core.strategy.getEngine('enhanced');
const result = strategy.analyze(scenario);
```

### 技能使用

```javascript
// 使用AI工具包
const aiToolkit = require('./skills/ai-toolkit');
// 使用提示工程
aiToolkit.prompt-engineering/scripts/quality-checker.ps1 ...

// 使用搜索工具包
const searchToolkit = require('./skills/search-toolkit');
// 使用Exa搜索
mcporter call 'exa.web_search_exa(query: "...")' ...

// 使用开发工具包
const devToolkit = require('./skills/dev-toolkit');
// 使用API开发
curl -X POST https://api.example.com/users -d '...' ...
```
