## Weather Skill

This skill provides weather information without requiring API keys.

### Features

- **Current weather**: Get current conditions for any location
- **Weather forecasts**: Receive short-term and long-term weather predictions
- **Multiple locations**: Check weather for different cities worldwide
- **Weather alerts**: Get notified about severe weather conditions

### Usage

When users ask about weather conditions, forecasts, or climate information, use this skill to provide accurate weather data.

### Examples

- "What's the weather like in Tokyo?"
- "Will it rain tomorrow in London?"
- "Show me the weather forecast for this week"
- "Is there any storm warning in my area?"

### Notes

- No API key required for basic weather functionality
- Supports multiple weather sources for reliability
- Provides both current conditions and forecast data
- Mobile-friendly formatting for different devices
