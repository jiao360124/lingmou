## 🏗️ 六层架构

```
┌─────────────────────────────────────────────────────────┐
│           感知层（监控）                                  │
│  - MetricsTracker - 指标追踪                             │
│  - 实时数据采集                                            │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│           预测层（风险评估）                              │
│  - RiskAssessmentEngine - 风险评估                        │
│  - 压力预测 + 成本预测 + 性能预测                          │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│           策略层（多方案生成）                            │
│  - StrategyEngine - 策略引擎                              │
│  - 4种策略类型                                            │
│  - 场景模拟                                               │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│           决策层（最优选择）                              │
│  - DecisionEngine - 决策引擎                              │
│  - 成本收益评分 + 风险评估 + 自我博弈                      │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│           执行层（Runtime）                               │
│  - RuntimeExecutor - 运行时执行器                         │
│  - 策略应用 + 实时监控                                    │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│           反馈层（指标追踪）                              │
│  - FeedbackCollector - 反馈收集器                         │
│  - 成功/失败记录 + 学习更新                               │
└─────────────────────────────────────────────────────────┘
```
