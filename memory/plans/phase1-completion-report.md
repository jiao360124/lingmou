## 🎯 核心特性

### 1. 策略类型
- **激进型** (AGGRESSIVE): 快速响应，高消耗，高收益
- **保守型** (CONSERVATIVE): 缓慢响应，低消耗，低风险
- **平衡型** (BALANCED): 中等响应，中等消耗，平衡收益
- **探索型** (EXPLORATORY): 尝试新方法，探索未知收益

### 2. 评分机制
```javascript
综合得分 = (收益总分 - 风险分数 × 风险权重) × 收益权重
```

**收益权重分布**:
- 成功率增益: 35%
- 成本减少: 25%
- 延迟改善: 20%
- 压缩改善: 10%
- 模型得分: 10%

**风险权重分布**:
- 成功率风险: 40%
- 成本比率: 30%
- 延迟比率: 20%
- 压缩等级: 10%
- 模型风险: 固定值

### 3. 约束检查
- 延迟约束: `context.constraints.maxDelay`
- 成本约束: `context.budgetConstraints.maxCost`
- 支持多个约束同时存在

### 4. 学习机制
- 实时记录策略选择
- 持久化到 JSON 文件
- 加载历史学习数据
- 支持策略模式进化
