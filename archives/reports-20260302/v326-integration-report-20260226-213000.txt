### 3. 技能整合

#### AI/LLM工具包（ai-toolkit）

**整合的技能**:
1. langchain → prompt-engineering
2. moltbook → moltbook
3. rag → rag

**功能描述**:
- **Prompt Engineering**: 提示模板、质量检查、优化建议
- **Moltbook**: AI Agent社区平台集成
- **RAG**: 检索增强生成知识库

**使用示例**:
```bash
# 提示工程
pe templates --use code.function

# Moltbook社区
.\api-client.ps1 -Action post -Content "..."

# RAG知识库
rag search "API调用"
```

#### 搜索工具包（search-toolkit）

**整合的技能**:
1. smart-search → exa-web-search
2. deepwiki → deepwiki

**功能描述**:
- **Exa Web Search**: AI驱动搜索（网络、代码、公司）
- **DeepWiki**: GitHub仓库文档查询

**使用示例**:
```bash
# Exa搜索
mcporter call 'exa.web_search_exa(query: "...")'

# DeepWiki
node ./scripts/deepwiki.js ask cognitionlabs/devin "..."
```

#### 开发工具包（dev-toolkit）

**整合的技能**:
1. api-dev → api-dev
2. database → database
3. sql-toolkit → sql-toolkit

**功能描述**:
- **API Development**: REST/GraphQL API开发
- **Database Manager**: 多数据库管理
- **SQL Toolkit**: SQLite、PostgreSQL、MySQL

**使用示例**:
```bash
# API开发
curl -X POST https://api.example.com/users -d '...'

# 数据库管理
"Connect to PostgreSQL database"

# SQL工具包
sqlite3 mydb.sqlite "SELECT * FROM users;"
```

## 🚀 使用指南

### 基本使用

```javascript
// 导入核心模块
const core = require('./core');

// 初始化
core.init({
  performance: { checkInterval: 60000 },
  memory: { checkInterval: 60000 },
  api: { trackInterval: 60000 }
});

// 使用监控
core.performanceMonitor.getCurrentMemoryUsage();

// 使用策略引擎
const strategy = core.strategy.getEngine('enhanced');
const result = strategy.analyze(scenario);
```
